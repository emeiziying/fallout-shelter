# 🏠 宿舍完整指南

## 🎯 **宿舍的核心作用**

宿舍是**人口扩张的关键**，没有宿舍就无法招募更多居民！

### 📊 **人口容量系统**
- **基础容量**: 3人 (游戏开始时的容量)
- **宿舍加成**: 每个宿舍每级提供 **+4人口位**
- **计算公式**: 总人口上限 = 基础容量(3) + 宿舍总容量

### 🏗️ **宿舍建造详情**
- **建造费用**: 材料×30 ⭐ **最便宜的建筑！**
- **建造时间**: 10秒
- **升级费用**: 材料×60 + 组件×5 (每级)
- **工人需求**: 0 (无需分配工人)
